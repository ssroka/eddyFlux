
6-hourly data for a patch from 
lat: 25  to 45
lon: 130 to 170


q_o is calculated using the SAM_qsatWater.m function which is based on Flatau,
Walko, and Cotton (1992): "Polynomial Fits to Saturation Vapor Pressure"
This is identical to the one used in the System for Atmospheric Modeling, version 6.10.8.

q_a and RH are calculated using SAM_psatWater.m which is based on Flatau,
Walko, and Cotton (1992): "Polynomial Fits to Saturation Vapor Pressure"
This is identical to the one used in the System for Atmospheric Modeling, version 6.10.8.
